What this extension does with your private data
  • It DOES send hostname of almost every website you're visiting to an external website (called " source ", in options you can choose which one, default is ipinfodb.com) to get its location information when you're visiting it for a first time.
  • It DOES save list of hostnames for all sites you've ever visited together with their location information and first visit time (feature called " cache ") for about one month.
  • It does NOT request source on every site visit or page load (since information gets retrieved from cache).
  • It does NOT request source or store in cache information about most popular websites, because information about them is already stored in extension itself (feature called " precache ")
  • It DOES ask Opera to use " custom flags ", if you set any, every time you switch to or open a page of a site from related country. If it is cached or not is decided by Opera and is out of control of this extension.
  • It does NOT send any data related to your browsing history to anyone apart from explained above (to source or custom flags ) and in separate section below (for statistics , if you enable it)
  • It uses unencrypted connections, meaning anyone with sufficient knowledge around you can "sniff" and copy or even modify all communicated data.
  • On each page it sets javascript object window['flag-button'] so other scripts and extensions could benefit from knowing location information.
  • Opera browser periodically checks for extension updates -- it is expected behaviour and you should be aware of it.
  • Any extension behaviour outside of features discussed here is a bug (either of extension or this privacy policy) and should be immediately reported to the developer.

Privacy policy concerning usage statistics

  • Participation in statistics collection is purely optional, off by default and no other functions of this extension depend on it
  • Usage statistics is designed to be as anonymous as possible -- no personal information is stored and no attempt is made to link separate pieces of data to each other or to individual users
  • Description of data that will be sent is available in relevant section in settings. In same place you can review the data you're sending and stop participating in sending statistics
  • Extension does NOT send individual hostnames or visiting times -- only "total numbers" are sent
  • Extension does NOT send preferences which may contain your private data
  • You CAN NOT change how often you send data, because statistics are based on assumption that everyone sends data with same frequency
  • You CAN NOT change the data itself, because it will undermine my belief in it
  • Statistics server DOES record current time, your IP and User-Agent string of your browser when you submit data - this is a standard feature of all websites. This information will NOT be used to identify you.
