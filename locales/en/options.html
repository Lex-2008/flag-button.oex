Popup window
Popup is a small window that you see when click extension button

Links list

Below you can change what links do you want to see at the bottom of the popup window. Each line contains link title and URL, separated by equal sign.
You can use several [placeholders] to customize URLs: [host] (domain name), [ip], [country], [region], [city], [zip] (zip/postal code), [lat] (latitude), [lng] (longitude), [tz] (time zone).

Custom icons
If you noticed an outdated icon, don't like the default "unknown flag" icon, or want to use another icon for some country, you can configure it here. On each line, type two-letter country code and custom icon URL, separated by equal sign. Note that you can use XX country code for "unknown flag" (default icon). For example, to change it back to European Union flag, add this line: XX=http://flag-button.tk/European-Union-Flag-16.png
Source
Every time you visit a new site, this extension sends query about its location to the selected below source site.
Warning! Nobody can guarantee that selected source site doesn't keep a history of your queries and doesn't save a list of all sites you have visited. In fact, most of sites in the Internet include banners, which owners have and most probably use this information.




For using ipinfodb.com source it is recommended to register (it's free) and enter provided access key. This will help to optimize server resources and remove the limit on parallel queries (Without using a key speed is limited to 1 request per second from all users of this extension. After registration the speed is limited to 1 request per second from you only).
