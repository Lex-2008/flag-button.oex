Особые флаги
Если Вы заметили устаревшую иконку флага, Вам не нравится "неизвестный флаг", используемый по умолчанию, или Вы хотите использовать другую иконку для какой-либо страны, Вы можете настроить это здесь. На каждой строке введите двухбуквенный код страны и адрес иконки, разделенные знаком равенства. Вы так же можете использовать XX в качестве кода страны для замены "неизвестного флага", используемого по умолчанию. Например, чтобы заменить его обратно на флаг Европейского Союза, добавьте такую строчку: XX=http://flag-button.tk/European-Union-Flag-16.png
Источник
Каждый раз, когда Вы открываете новый сайт, это расширение отправляет запрос о его расположении на выбранный ниже сайт-источник.
Внимание! Никто не может гарантировать, что сайт-источник не ведет историю запросов и не хранит список всех сайтов, на которые Вы заходили. На самом деле, на большинстве интернет-сайтов есть рекламные блоки, хозяева которых обладают и наверняка пользуются этой информацией.




Для использования источника ipinfodb.com рекомендуется бесплатно зарегистрироваться и указать полученный ключ доступа. Это поможет оптимизировать ресурсы сервера и снять ограничение на количество параллельных запросов (Без использования ключа ограничение скорости в 1 запрос в секунду делится между всеми пользователями расширения. После регистрации скорость ограничена 1 запросом в секунду только от Вас).

Оффлайн режим

При включении этого режима запросы к сайту-источнику не будут отправляться, и будет отображаться информация только об уже посещенных сайтах
Общие
Отключение следующей опции поможет избежать мигания иконки при переключении между вкладками, но на некоторых страницах (например, на экспресс-панели) Вы будете видеть флаг, оставшийся с предыдущей страницы.


Statistics

Enabling this feature will send the following information to the developer:
  • Extension version (so I know when autoupdate works)
  • Size of cache and usage of precache and its different parts (cache is a memory that keeps location information for sites that you've visited. Precache is a similar thing, but collected while building the extension with looking at alexa "top million sites" list and internal statistics of requests to flag-button.tk. These numbers will show me how hard should I think about optimizing them)
  • All preferences except those that might have personal information (ipinfodb.com access key, links list and custom icons - these preferences won't be sent. Knowing how users change preferences will give me hints how settings should be configured by default)
